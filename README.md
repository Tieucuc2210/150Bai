# Bài 1 : Tính Giá Trị Của biểu thức

Cho biểu thức A(x) = x^3 + 3x^2 + x + 1 Với giá trị của x được nhập từ bàn phím, tính và in ra giá trị của biểu thức trên

Input Format

Số nguyên x

Constraints

-10^5≤x≤10^5

Output Format

In ra kết quả cùa biểu thức

Sample Input 0


```pre
2
```


Sample Output 0

<pre>
23	
</pre>
